# 👤 Guest Role

**Access Level:** PUBLIC
**Purpose:** Default role for new or anonymous users.

## Access Profile

The Guest role provides a baseline reading experience. It includes core accessibility and UI features while withholding advanced stats, navigation, and experimental tools.

## Strategic Purpose

The Guest role is designed to be the "free tier" of the application, allowing users to experience the core value proposition (reading canonical fairy tales with accessibility tools) before committing to a subscription or joining a program.
