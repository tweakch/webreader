# 🧪 Tester Role

**Access Level:** INTERNAL
**Purpose:** QA and beta users with access to experimental features.

## Access Profile

The Tester role provides the most comprehensive feature set outside of the Admin role. It includes all experimental features in the "Gen Alpha" category and critical debug tools.

## Included Features (Adds to Subscriber)

| Feature | Category |
|---|---|
| [Read-Along](../features/read-along.md) | Gen Alpha |
| [Illustrations](../features/illustrations.md) | Gen Alpha |
| [Child Profile](../features/child-profile.md) | Gen Alpha |
| [Story Quiz](../features/story-quiz.md) | Gen Alpha |
| [Text-to-Speech](../features/text-to-speech.md) | Advanced Reading |
| [Simplified UI](../features/simplified-ui.md) | Appearance |
| [Debug Badges](../features/debug-badges.md) | Debug |
| [Error Page Simulator](../features/error-page-simulator.md) | Debug |

## Strategic Purpose

The Tester role is the gateway to the product's future. It allows internal team members, beta testers, and selected power-users to evaluate unreleased features in real-world scenarios before they are moved to the Subscriber or Guest tiers.
