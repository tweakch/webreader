# @tweakch/collection-grimm-klassiker

Fifteen canonical Brothers Grimm fairy tales — the classics that belong in any household library. This collection expands the Top 5 to the core canon that most German speakers grew up with.

## Install

```
@tweakch:registry=https://npm.pkg.github.com
```

```bash
npm install @tweakch/collection-grimm-klassiker
```

After installation the collection is picked up automatically at build time by webreader's Vite plugin and appears as its own source ("Grimm Klassiker") in the sidebar.
