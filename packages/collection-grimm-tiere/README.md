## Install

```
@tweakch:registry=https://npm.pkg.github.com
```

```bash
npm install @tweakch/collection-grimm-tiere
```

After installation the collection is picked up automatically at build time by webreader's Vite plugin and appears as its own source ("Grimm Tiermärchen") in the sidebar.
