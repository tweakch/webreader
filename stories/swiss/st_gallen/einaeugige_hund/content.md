# 127. Der einäugige Hund

Nahe bei Räsis, an der Sarbrücke, begegnet dem späten Wanderer ein großer, schwarzer Hund, der nur ein Auge hat. Dieses sitzt mitten auf der Stirn, ist aber so groß wie eine "Butzenscheibe".

Heinrich Hilty
