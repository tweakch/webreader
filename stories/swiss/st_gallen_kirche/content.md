# 16. Die Kirche

In Mörschwil soll nach einer alten Überlieferung eine Bruderkapelle gestanden haben. Im Jahre 1494 wurden dort in einer von altem Mauerwerk umschlossenen Stätte etliche Totengerippe gefunden. Der Ort bekam einen Bildstock, zu dem man wallfahrtete, hierauf eine Kapelle, dann die Kirche.
